## 🌟 核心特性 (Features)

- **💎 极简暗黑毛玻璃 UI (Glassmorphism)**: 2026 顶级视觉美学，动效丝滑，适配各种端侧。
- **🎙️ 语音/视觉双模态 (Multi-modal)**: 支持 Web Speech API 语音输入与摄像头抓拍，全方位理解你的任性。
- **⚡ 零成本运营 (User-Managed API)**: 用户自带 OpenAI/Gemini API Key，开发者服务器 0 算力支出。
- **🕸️ 静默数据图谱收割 (Silent Data Harvester)**: 全自动、无感抓取用户交互数据并沉淀到 Supabase 云端，构建人类创意图谱。
- **📱 移动端 APP 化 (PWA Ready)**: 完整支持 PWA 规范，一键“添加到主屏幕”秒变原生手机 APP。
- **🧪 本地免 API 测试模式 (Mock Mode)**: 留空 API Key 即可启动仿真环境，极速验证产品逻辑。

## 🚀 极速部署 (Quick Start)

项目采用 **Antigravity (极简/无冗余)** 架构，纯前端实现，双击即用。

### 1. 本地运行
直接在浏览器中打开 `index.html` 即可开始你的任性之旅。

### 2. 云端发布
将本项目推送到 GitHub 后，使用 **Netlify** 或 **Vercel** 关联仓库，实现秒级上线。

### 3. PWA 安装
在手机端打开部署后的链接，点击浏览器菜单中的“**安装应用**”或“**添加到主屏幕**”，即可在桌面获得“我就要”独立图标。
